## DC9 是什么机房，为什么单独拿出来说

搬瓦工洛杉矶有好几个机房，DC2、DC4、DC6、DC9，编号不一样，线路也不一样。**DC9 的编号是 USCA\_9**，目前用的是 AMD EPYC 处理器加 NVMe SSD 存储，硬件这一轮是真升级了。

线路方面，DC9 走的是三网回程优化——电信 CN2 GIA、联通 Premium（AS10099）、移动 CMIN2，三家运营商各走自己的高端专线。晚高峰的时候，这三条线路同时稳住，比起只优化电信一家的方案覆盖面更广。

到大陆的延迟，DC9 大概在 130–160ms 之间，这是洛杉矶到国内的物理距离决定的，比香港（20–30ms）高不少，但比普通线路晚高峰乱掉连 VPN 都走不通要稳多了。

一句话定位：DC9 是搬瓦工洛杉矶机房里线路最好的一个，但它不是独立套餐——而是多个套餐里**可以选择迁移进去的目标机房**。
